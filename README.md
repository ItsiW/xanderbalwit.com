# xanderbalwit.com

Personal website for Xander Balwit (Alexandra Balwit). Single-page portfolio site.

## Tech Stack

- **Framework:** Astro 5 (static site generator)
- **Styling:** Tailwind CSS 4 (using `@theme` for custom colors in `src/styles/global.css`)
- **Hosting:** Cloudflare Workers (via Workers Builds, auto-deploys on push to `main`)
- **Domain:** xanderbalwit.com (registered on Namecheap, nameservers pointed to Cloudflare)
- **Config:** `wrangler.jsonc` tells Cloudflare to deploy the `./dist` directory as static assets

## Development

```sh
npm install
npm run dev        # localhost:4321
npm run build      # outputs to ./dist
```

## Deployment

Push to `main` on GitHub → Cloudflare Workers Builds auto-builds and deploys.

- GitHub repo: github.com/ItsiW/xanderbalwit.com
- Cloudflare dashboard: dash.cloudflare.com → Workers & Pages → xanderbalwit
- Build command: `npm run build`
- Deploy command: `npx wrangler deploy`
- Output directory: `./dist` (configured in `wrangler.jsonc`)

## Project Structure

```
public/
  xander.jpg          # Hero photo
  favicon.svg         # Yellow X on red background
scripts/
  fetch-og-images.mjs # Fetches og:image URLs from article links, saves to og-images.json
src/
  components/
    ArticleCard.astro  # Writing card with optional thumbnail image
    Footer.astro       # Site footer
    Hero.astro         # Reusable hero (not currently used, hero is inline in index)
    Nav.astro          # Top nav with anchor links
    ProjectCard.astro  # Project card
  data/
    og-images.json     # Cached OG image URLs for all articles (generated by fetch script)
  layouts/
    BaseLayout.astro   # Base HTML layout
  pages/
    index.astro        # THE site — single page with all sections
  styles/
    global.css         # Tailwind config, custom color palette
wrangler.jsonc         # Cloudflare Workers deployment config
```

## Adding/Updating Articles

1. Edit `src/pages/index.astro` — add an `<ArticleCard>` in the appropriate year section
2. Run `node scripts/fetch-og-images.mjs` to fetch the OG image for the new URL (or manually add to `src/data/og-images.json`)
3. Push to `main`
